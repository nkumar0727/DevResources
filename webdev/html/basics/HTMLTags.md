# HTML Tags Overview

## Tag Usages

For **bold** text with Semantic meaning:
```html
<strong>Text</strong>
```

For *emphasized* text with Semantic menaing:
```html
<em>Text</em>
```

For grouping stuff (block level element):
```html
<div>
  <h1>Text</h1>
  <p>This is a paragraph</p>
</div>
```

For grouping stuff (inline element)
```html
<span>
  <h1>Text</h1>
  <p>This is a paragraph</p>
</span>
```
